eChartz

eChartz is a simple web application for making dynamic 2D charts. Users can enter their data or copy/paste their own excel data into the table and generate various 2D charts. This project is built using NodeJs along with MVC theory. Mongodb is used to store the multidimensional array data structure of the table content. The live stock graph fetches the stock quota data from Yahoo finance API. Pug (Jade), Bootstrap, jQuery are applied to produce a professional looking, animated and iterative website.

  • Date: August 2016
  • Link: www.echartz.com

CodeCore For Hire

CodeCore for Hire is a Ruby on Rails web application where employers can search for potential candidates in the tech industry. The 72-Hour hackathon project was built utilizing Agile methodologies and Github for version control and collaboration. It includes features like Uploading files, User authentication, Password reset, Cloud storage AWS3 etc. For the front end, HTML5, CSS3, JavaScript and Bootstrap were used to produce a professional and stylized interface. In addition, AJAX and jQuery were applied to improve user experience.

  • Date: July 2016
  • Link: https://codecoreforhire.herokuapp.com
